Schema WOKDEX

Modelo de metadados pedagógicos validado por JSON Schema que organiza exercícios de programação em três camadas semânticas.

Arquitetura

Três Camadas do Schema

O WOKDEX organiza os metadados em camadas com responsabilidades distintas.

Camada 1

📋 Descritiva

Informações gerais do exercício: título, nível (CS1/CS2/CS3), dificuldade, complexidade temporal e tags.

name slug difficultyLevelId timeComplexity tags origin
Camada 2

🧠 Pedagógica

Competências cognitivas necessárias (skills), editorial para o professor e mapeamento curricular.

skills editorial successmsg helpTip thematicSkills
Camada 3

🎯 Avaliativa

Cenários de teste tipados por intenção pedagógica, com níveis hierárquicos e dicas formativas.

testScenarios testType level helpTip HCS
Tipologia Estendida

Quatro Tipos de Teste

Cada cenário de teste é classificado segundo sua intenção pedagógica.

Tipo Visível? Descrição Exemplo de Uso
SAMPLE ✅ Sim Testes correspondentes aos exemplos do enunciado, utilizáveis para depuração local antes da submissão. Entrada e saída do PDF do exercício
TDD ❌ Não Teste funcional secreto que valida um comportamento específico não revelado ao estudante. Casos de borda, valores negativos, vetores vazios
FALSE GREEN ❌ Não Teste projetado para invalidar soluções acidentalmente corretas — detecta misconceptions (MC³). Divisão 19/6: expõe int vs double
PERFORMANCE ❌ Não Avalia complexidade algorítmica (tempo, memória), separando-a da corretude lógica. N=10⁵: O(N²) excede tempo, O(N log N) passa

Subtipos de TDD_FALSE_GREEN

Subtipo Misconception Exemplo Nível
TYPE Tipo de dado inadequado int em vez de double para divisão CS1
FORMAT Formatação de saída incorreta println em vez de printf("%.2f") CS1
STRATEGY Estratégia algorítmica insuficiente BFS em grafo ponderado em vez de Dijkstra CS2–CS3

    skills:

⚠️ Comportamento FALSE_GREEN

Misconception: int em vez de double

O aluno declara variáveis como int:

int a = 10, b = 2;
double r = (double)(a/b); // = 5.00 ✅

int a = 19, b = 6;
double r = (double)(a/b); // = 3.00 ❌
// Esperado: 3.17

O cast é aplicado após a divisão inteira. O cenário TDD_FALSE_GREEN usa entradas como (19,6) e (1,3) que expõem esse erro.

💡 Feedback WOKDEX

"Erro na declaração de tipo de variável. Suas variáveis são inteiras, mas a divisão pode gerar resultado decimal. Verifique se está usando double/float."

Índice de Confiança

Hint Confidence Score (HCS)

Modula a emissão do feedback com base na consistência dos erros observados.

HCS = falhascenário / totalcenário

Faixa HCS Ação do Motor
Alta Confiança 1.0 (todos falharam) Emite helpTip diretamente ao estudante
Confiança Moderada ≥ 0.5 Emite "Possível causa: {helpTip}"
Baixa Confiança < 0.5 Delega a dica genérica ou módulo de IA
