Contribuições

O que o WOKDEX Resolve

Quatro contribuições para transformar a avaliação automática em diagnóstico pedagógico.

C1

Schema de Metadados

Modelo estruturado (YAML validado por JSON Schema) que formaliza atributos pedagógicos para exercícios de programação.

C2

Tipologia de Testes

Categoriza cenários por intenção pedagógica: SAMPLE, TDD, TDD_FALSE_GREEN e PERFORMANCE.

C3

Corpus de Referência

Exercícios reais adaptados para CS1–CS3, com cenários deliberadamente projetados para detectar misconceptions.

C4

Hint Confidence Score

Índice que modula a confiabilidade do feedback com base na taxa de falha intra-cenário (HCS).
