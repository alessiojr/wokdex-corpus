# Suco de Acerola Espacial

Na estação espacial WOK-42, a tripulação cultiva **Acerola Cósmica** em estufas pressurizadas. Após cada colheita, as frutas são processadas no extrator de bordo para produzir suco, que é dividido igualmente entre os tripulantes.

Sabe-se que:

*   Cada unidade de fruta produz exatamente **50 ml** de suco.
*   O suco total é dividido igualmente entre todos os tripulantes.

Dado o número de tripulantes e a quantidade de frutas colhidas, determine o volume de suco, **em litros**, que cada tripulante receberá.

## Entrada

A entrada contém vários casos de teste, cada um em uma única linha.

Cada linha contém dois inteiros $N$ e $F$ ($1 \leq N \leq 10^3$, $1 \leq F \leq 10^3$), representando o número de tripulantes e a quantidade de frutas colhidas, respectivamente.

O final da entrada é indicado por uma linha contendo `0 0`.

## Saída

Para cada caso de teste, imprima uma linha contendo um número real com exatamente **duas casas decimais**, representando o volume em litros que cada tripulante receberá.

## Exemplos

| Entrada | Saída |
|:--- |:--- |
| 1 1 | 0.05 |
| 5 431 | 4.31 |
| 101 330 | 0.16 |
| 0 0 | |
