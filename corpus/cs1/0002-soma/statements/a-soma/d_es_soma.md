# Aprenda a Sumar

Vamos a crear un programa simple para sumar dos números.
Siga los pasos a continuación:

1. Lea un número entero del teclado (lo llamaremos X).
2. Lea otro número entero del teclado (lo llamaremos Y).
3. Calcule el valor de X + Y.
4. Muestre este valor en la pantalla.

## Entrada
- Un número entero $X$ ($-10^4 \leq X \leq 10^4$).
- Un número entero $Y$ ($-10^4 \leq Y \leq 10^4$).

## Salida
- El resultado de la suma.

## Ejemplos
| Entrada | Salida |
| :--- | :--- |
| 5<br>2 | 7 |

| Entrada | Salida |
| :--- | :--- |
| 100<br>200 | 300 |
