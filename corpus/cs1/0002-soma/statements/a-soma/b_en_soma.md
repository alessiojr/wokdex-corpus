# Simple Sum

Create a program that reads two integer values and presents the result of the sum between them.

## Input
The input contains two integers $A$ and $B$, one on each line.
*Domain: $-10^4 \leq A, B \leq 10^4$*

## Output
Print a single integer value corresponding to the sum of $A$ and $B$.

## Examples
| Input | Output |
| :--- | :--- |
| 10<br>20 | 30 |

| Input | Output |
| :--- | :--- |
| 5<br>3 | 8 |
