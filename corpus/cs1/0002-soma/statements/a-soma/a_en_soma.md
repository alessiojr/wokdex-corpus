# Complex Addition Theorem

In the field of real and complex analysis, the addition operation is fundamental. Consider two numerical bodies in a one-dimensional vector space.

To perform the $\sum$ operation, we must disregard the imaginary part (assuming $i=0$) and focus purely on the real scalar magnitude of the components. Do not worry about partial derivatives or line integrals at this moment.

Your task is to simplify the equation $f(x,y) = x + y$ for two discrete values provided by the linear input system.

## Input
The input consists of two integers $X$ and $Y$.
*Domain: $-10^4 \leq X, Y \leq 10^4$*

## Output
A single integer representing the result of the addition operation.

## Examples
| Input | Output |
| :--- | :--- |
| 2<br>7 | 9 |

| Input | Output |
| :--- | :--- |
| 10<br>5 | 15 |
