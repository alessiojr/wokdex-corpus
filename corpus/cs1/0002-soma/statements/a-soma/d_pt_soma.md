# Aprenda a Somar

Vamos criar um programa simples para somar dois números.
Siga os passos abaixo:

1. Leia um número inteiro do teclado (chamaremos de X).
2. Leia outro número inteiro do teclado (chamaremos de Y).
3. Calcule o valor de X + Y.
4. Mostre esse valor na tela.

## Entrada
- Um número inteiro $X$ ($-10^4 \leq X \leq 10^4$).
- Um número inteiro $Y$ ($-10^4 \leq Y \leq 10^4$).

## Saída
- O resultado da soma.

## Exemplos
| Entrada | Saída |
| :--- | :--- |
| 5<br>2 | 7 |

| Entrada | Saída |
| :--- | :--- |
| 100<br>200 | 300 |
