# Learn to Sum

Let's create a simple program to sum two numbers.
Follow the steps below:

1. Read an integer from the keyboard (let's call it X).
2. Read another integer from the keyboard (let's call it Y).
3. Calculate the value of X + Y.
4. Show this value on the screen.

## Input
- An integer $X$ ($-10^4 \leq X \leq 10^4$).
- An integer $Y$ ($-10^4 \leq Y \leq 10^4$).

## Output
- The result of the sum.

## Examples
| Input | Output |
| :--- | :--- |
| 5<br>2 | 7 |

| Input | Output |
| :--- | :--- |
| 100<br>200 | 300 |
