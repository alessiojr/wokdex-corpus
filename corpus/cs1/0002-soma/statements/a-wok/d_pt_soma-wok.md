# Computador de Bordo

Olá, Cadete! Vamos programar o computador de bordo para calcular a rota.
Você tem a distância do Trecho 1 e a distância do Trecho 2. Precisa apenas somá-las.

**Instruções:**

1. Receba a distância do primeiro planeta.
2. Receba a distância do segundo planeta.
3. Some as duas distâncias.
4. Imprima o total no painel.

## Entrada
- Um número inteiro $D1$ (distância 1).
- Um número inteiro $D2$ (distância 2).
*Domínio: $-10^4 \leq D1, D2 \leq 10^4$*

## Saída
- O total da viagem ($D1 + D2$).

## Exemplos
| Entrada | Saída |
| :--- | :--- |
| 5<br>2 | 7 |

| Entrada | Saída |
| :--- | :--- |
| 100<br>200 | 300 |
