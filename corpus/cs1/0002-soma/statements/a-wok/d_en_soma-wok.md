# Onboard Computer

Hello, Cadet! Let's program the onboard computer to calculate the route.
You have the distance for Leg 1 and the distance for Leg 2. You just need to add them up.

**Instructions:**

1. Receive the distance of the first planet.
2. Receive the distance of the second planet.
3. Sum the two distances.
4. Print the total on the display.

## Input
- An integer $D1$ (distance 1).
- An integer $D2$ (distance 2).
*Domain: $-10^4 \leq D1, D2 \leq 10^4$*

## Output
- The total trip ($D1 + D2$).

## Examples
| Input | Output |
| :--- | :--- |
| 5<br>2 | 7 |

| Input | Output |
| :--- | :--- |
| 100<br>200 | 300 |
