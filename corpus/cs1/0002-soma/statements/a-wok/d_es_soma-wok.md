# Computadora de a Bordo

¡Hola, Cadete! Vamos a programar la computadora de a bordo para calcular la ruta.
Tienes la distancia del Tramo 1 y la distancia del Tramo 2. Solo necesitas sumarlas.

**Instrucciones:**

1. Reciba la distancia del primer planeta.
2. Reciba la distancia del segundo planeta.
3. Sume las dos distancias.
4. Imprima el total en el panel.

## Entrada
- Un número entero $D1$ (distancia 1).
- Un número entero $D2$ (distancia 2).
*Dominio: $-10^4 \leq D1, D2 \leq 10^4$*

## Salida
- El total del viaje ($D1 + D2$).

## Ejemplos
| Entrada | Salida |
| :--- | :--- |
| 5<br>2 | 7 |

| Entrada | Salida |
| :--- | :--- |
| 100<br>200 | 300 |
