# Calculadora de Fundição

Você é um aprendiz de metalurgia e precisa calcular quanto material vai usar.
Para saber o total, basta somar o peso do primeiro metal com o peso do segundo metal.

**O que você deve fazer:**

1. Leia o primeiro número (Peso do Metal 1).
2. Leia o segundo número (Peso do Metal 2).
3. Faça a conta de adição (+).
4. Imprima o resultado.

## Entrada
- Um número inteiro $A$ ($-10^4 \leq A \leq 10^4$).
- Um número inteiro $B$ ($-10^4 \leq B \leq 10^4$).

## Saída
- Um número inteiro mostrando a soma $A + B$.

## Exemplos

| Entrada | Saída |
| :--- | :--- |
| 5<br>2 | 7 |

| Entrada | Saída |
| :--- | :--- |
| 100<br>200 | 300 |

*Dica: Use `cin >> a >> b;` em C++ ou `input()` em Python.*
