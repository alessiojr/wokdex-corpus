# Foundry Calculator

You are a metallurgy apprentice and need to calculate how much material you will use.
To find the total, simply add the weight of the first metal to the weight of the second metal.

**What you must do:**

1. Read the first number (Weight of Metal 1).
2. Read the second number (Weight of Metal 2).
3. Do the addition (+) calculation.
4. Print the result.

## Input
- An integer $A$ ($-10^4 \leq A \leq 10^4$).
- An integer $B$ ($-10^4 \leq B \leq 10^4$).

## Output
- An integer showing the sum $A + B$.

## Examples
| Input | Output |
| :--- | :--- |
| 5<br>2 | 7 |

| Input | Output |
| :--- | :--- |
| 100<br>200 | 300 |

*Tip: Use `cin >> a >> b;` in C++ or `input()` in Python.*
