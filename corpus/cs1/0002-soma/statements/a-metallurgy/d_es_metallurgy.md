# Calculadora de Fundición

Usted es un aprendiz de metalurgia y necesita calcular cuánto material va a usar.
Para saber el total, basta sumar el peso del primer metal con el peso del segundo metal.

**Lo que debe hacer:**

1. Lea el primer número (Peso del Metal 1).
2. Lea el segundo número (Peso del Metal 2).
3. Haga la cuenta de suma (+).
4. Imprima el resultado.

## Entrada
- Un número entero $A$ ($-10^4 \leq A \leq 10^4$).
- Un número entero $B$ ($-10^4 \leq B \leq 10^4$).

## Salida
- Un número entero mostrando la suma $A + B$.

## Ejemplos
| Entrada | Salida |
| :--- | :--- |
| 5<br>2 | 7 |

| Entrada | Salida |
| :--- | :--- |
| 100<br>200 | 300 |

*Consejo: Use `cin >> a >> b;` en C++ o `input()` en Python.*
