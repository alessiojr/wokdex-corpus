

# Olá mundo

![Hello Word](0001-hello-world.png "Hello Word"){width=150px align=right}

Este é um clássico exercício de iniciação à linguagem de computação usado por várias civilizações. 

Para iniciar sua jornada pelo **Mundo do Código** Diga "Olá mundo". Mas como nossa jornada envolve várias linguagens vamos começar dizendo isso em inglês. (**Hello world!**)

## Entrada

O Programa não tem uma entrada ele apenas tem uma saída. 

## Saída

A saída deve ser apenas uma string contendo "**Hello world!**"

## Exemplos

| Entradas |       Saídas |
|       --:|           --:|
|          | Hello world! |
