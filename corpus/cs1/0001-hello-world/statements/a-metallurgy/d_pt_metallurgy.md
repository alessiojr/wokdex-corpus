
# Olá Forja do Código

Bem-vindo à **Forja do Código**, onde transformamos ideias em soluções através do fogo da programação! 

Como todo metalúrgico que se preza, você deve começar sua jornada dominando os fundamentos. Na metalurgia, o primeiro passo é sempre aquecer o forno e preparar a matéria-prima. Na programação, o primeiro passo é dominar a arte de fazer o computador "falar".

Para iniciar sua jornada pelo **Mundo do Código**, você deve fazer o computador cumprimentar o mundo, assim como um metalúrgico experiente cumprimenta seus colegas de forja. Mas como nossa jornada envolve várias linguagens e culturas, vamos começar dizendo isso em inglês: **"Hello world!"**

## Entrada

O programa não possui entrada - ele é como um forno que já está aquecido e pronto para trabalhar.

## Saída

A saída deve ser apenas uma string contendo **"Hello world!"** - o cumprimento universal que todo programador aprende a dominar, assim como todo metalúrgico aprende a controlar a temperatura do forno.
