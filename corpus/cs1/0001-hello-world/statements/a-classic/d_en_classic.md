# Hello world!

![Hello Word](0001-hello-world.png "Hello Word"){width=150px align=right}

This is a classic exercise in introducing the computer language used by various civilizations.

To start your journey through **World of Kode** say "Hello world!". But as our journey involves several languages, let's start by saying this in English. (**Hello world!**)

## Input

The program does not have an entry, it only has an exit.

## Output

The output must be just a string containing "**Hello world!**"
