# Cartas

Sendo 3 valores informados onde obrigatoriamente 2 são iguais, a resposta final sempre será aquele valor que é diferente do par principal.
Basta fazer três comandos `if` cruzados verificando cada hipótese:
- Se $A$ for igual a $B$, a resposta é $C$.
- Se $A$ for igual a $C$, a resposta é $B$.
- Se nenhuma delas for verdade, então obrigatoriamente $B$ é igual a $C$, logo a resposta é $A$.
